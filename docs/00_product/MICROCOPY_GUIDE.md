# Microcopy Guide

**Project:** In‑Play Predictor (CryptoScreener‑X) — ML + LLM  
**Date:** 2026-01-24

---


## Tone
Friendly, concise, trader-native. No hype. No predictions framed as certainties.

## Status labels (examples)
- TRADEABLE: “Tradable now”
- Tradeable soon: “Likely tradable soon”
- WATCH: “Watchlist”
- Hot but Dirty: “Hot, but risky flow”
- TRAP: “Trap risk”
- DEAD: “Low edge / high costs”
- DATA_ISSUE: “Data issue — don’t trust signal”

## Headline patterns
- “{SYM}: flow surge + tight spread — tradable soon.”
- “{SYM}: volatility up, but toxicity rising — hot but dirty.”
- “{SYM}: spread widened, impact high — dead for now.”
- “{SYM}: book stale — data issue.”

## Forbidden phrasing
- “Guaranteed”, “will pump”, “sure breakout”, “free money”
- Any new numbers not present in input JSON
