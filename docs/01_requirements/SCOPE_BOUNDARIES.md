# Scope Boundaries

**Project:** In‑Play Predictor (CryptoScreener‑X) — ML + LLM  
**Date:** 2026-01-24

---


## In scope (v1)
- Market data ingestion (WS) + feature engine + ML inference + ranking + alerts
- Recording + deterministic replay
- Basic dashboards/logging

## Out of scope (v1)
- Automated execution / order management
- Cross-exchange routing
- Full web UI (optional lightweight feed is OK)

## Future scope (v2+)
- Execution simulator w/ queue position
- Strategy plugins (pattern detection) on top of in-play engine
